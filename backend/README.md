# Backend folder

This directory is reserved for backend code (APIs, server-side logic, workers).

Suggested structure:

- `api/` - REST endpoints or route handlers
- `lib/` - shared server utilities
- `migrations/` - database migrations
- `README.md` - this file

Place server-side code here. If you're using `server.js` at the project root, consider moving or referencing backend modules from this folder.
