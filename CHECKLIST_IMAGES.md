## 🎨 DESIGN SPECIFICATIONS REFERENCE

### SVG Files (Use for illustrations & icons)
```
Format: SVG (Scalable Vector Graphics)
Size: 10-100KB per file
Dimensions: As specified in checklist above
Tools: Adobe Illustrator, Figma, Inkscape
Style: Clean, modern, professional
Color: Use consistent brand colors
```

### PNG Files (Use for UI mockups)
```
Format: PNG with transparency
Size: 100-300KB per file
Dimensions: Exact size specified (e.g., 800×600px)
DPI: 72 DPI (web standard)
Tools: Photoshop, Figma, or screenshot tools
```

### JPG Files (Use for photos)
```
Format: JPG (Progressive)
Size: 50-150KB per file
Dimensions: Exact size specified (e.g., 600×400px)
Quality: 75-85% compression
Tools: Camera, stock photos, Photoshop
```

## 🔍 TESTING AFTER UPLOAD

After adding each image:

1. **Check File Exists**
   ```
   Windows Explorer: D:\IncoziProject\Incozi\assets\images\...\filename.svg
   Should show the file you just added
   ```

2. **Test in Browser**
   ```
   Go to: http://localhost:3000/pages/[service].html
   Look for the image
   If missing, check DevTools (F12) Console for 404 errors
   ```

3. **Verify File Name**
   ```
   Image file must match EXACTLY:
   - Lowercase (inc-hero.svg, NOT Inc-Hero.svg)
   - Hyphens only (inc-hero, NOT inc_hero)
   - Correct extension (inc-hero.svg, NOT inc-hero.SVG)
   ```
